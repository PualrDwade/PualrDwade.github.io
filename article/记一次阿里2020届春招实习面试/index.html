阿里2020届春招实习面试

面试官给我的感觉就像是个小哥哥,很亲近也很耐心,整个过程还算顺利,一共面了有1个小时20分钟,还是有比较多的地方没有回答好,可能是还是比较紧张的原因,很简单的笔试题居然经过提醒之后才完成解答QAQ,主要的内容如下:

项目

架构,分工,实现,技术难点

操作系统

  1. 进程线程区别
  2. 虚拟内存管理(分段分页)
  3. linux常见命令有哪些

数据结构

  1. 两个栈实现队列
  2. 两个队列实现栈
  3. 二叉树的深度遍历，广度遍历
  4. 广度遍历代码实现
  5. 二叉搜索树原理
  6. 平衡二叉树是什么
  7. 二叉树平衡算法有哪些
  8. B树,B+树
  9. 红黑树

数据库

  1. innodb默认隔离级别
  2. 如何实现可重复读(MVCC)
  3. select * from table in share mode 加S锁
  4. select * from table for update 加X锁
  5. a,b,c三个字段联合索引,select * from table where a = x and b = y;索引是否生效,where a = x and c = y 呢?

网络

  1. http响应过程,从输入url到显示页面经历了什么
  2. DNS解析,端口,协议(upd,tcp)
  3. 三次握手,四次挥手过程
  4. 四次挥手客户端为什么要进入TIME-WAIT等待最大报文存活时间,而不是直接断开连接
  5. 应用层协议有哪些

java

  1. 原始类型,包装类型区别
  2. 装箱,拆箱过程
  3. 把空的包装类型赋值给原始类型会怎么样
  4. Integer底层怎么实现的
  5. == equals()区别
  6. hashcode()有什么用
  7. 为什么重写eq必须重写hashcode,
  8. 如何正确重写hashcode()
  9. LinkHashMap,TreeMap,HashMap
  10. 如果要保存元素插入顺序,用上面的哪几个
  11. HashMap如何解决哈希冲突

框架

  1. IOC,AOP概念
  2. Spring是怎么实现IOC和AOP的
  3. Spring的声明式事务如何实现的

设计模式

你知道的JDK用了哪些设计模式

笔试题

  1. 用递归实现字符串反转
  2. 找出一个第K大元素

校园经历

  1. 拿了什么奖,什么奖学金,有没有上线的项目
  2. 博客等等...
